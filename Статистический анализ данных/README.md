# Статистический анализ данных сервиса аренды самокатов GoFast
_______________________
  
  
    
##  О проекте
Проект представляет собой комплексный статистический анализ данных сервиса аренды самокатов GoFast. Включает обработку данных, исследовательский анализ, проверку гипотез и разработку рекомендаций для бизнеса.

###  Бизнес-модель сервиса:
- **Без подписки**:
  - Стоимость минуты: 8 ₽
  - Старт поездки: 50 ₽
  - Абонентская плата: 0 ₽
  
- **С подпиской Ultra**:
  - Стоимость минуты: 6 ₽
  - Старт поездки: 0 ₽
  - Абонентская плата: 199 ₽/мес

##  Цели проекта
1. **Аналитическая**: Провести исследование поведения пользователей и их предпочтений
2. **Бизнес-ориентированная**: Проверить гипотезы для оптимизации бизнес-процессов
3. **Прогностическая**: Оценить эффективность маркетинговых кампаний
----------


   
##  Ключевые результаты
  
- **Общая выручка**: 3,878,641 ₽
- **Распределение подписок**: Free (54.4%) vs Ultra (45.6%)
- **Топ-3 города**: Пятигорск, Екатеринбург, Ростов-на-Дону
- **Целевая аудитория**: Пользователи 22-28 лет
- **Пользователи с подпиской**:
  - Генерируют больше выручки (+27%)
  - Тратят на 15% больше времени на поездки
  - Проезжают оптимальное расстояние для износа самокатов (≤ 3130 м)



##  Проверенные гипотезы

###  Гипотеза 1: Время поездок
- **Статус**: Подтверждена
- **Вывод**: Пользователи с подпиской тратят на 15% больше времени

###  Гипотеза 2: Оптимальное расстояние
- **Статус**: Подтверждена  
- **Вывод**: Среднее расстояние ≤ 3130 м (оптимально для самокатов)

###  Гипотеза 3: Сравнение выручки
- **Статус**: Подтверждена
- **Вывод**: Выручка от подписчиков Ultra выше на 27%



##  Распределения и вероятности

- **Вероятность успеха**: 95%
